## Ciclo while:
Es una estructura que permite repetir 
Su bloque de codigo, **MIENTRAS QUE**
su condicional sea verdadera
-> si la condicioal tiene valor de 
verdad false, ya dejan de repetirse 
las instrucciones del bloque

### sintaxis :

```
while condicional:
INSTRUCCION 1
INSTRUCCION 2 ...
```
